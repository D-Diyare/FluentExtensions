List Extensions
Extensions to numeric data types such as: int, double, decimal, byte...

ToImmutableList

Changes a list from mutable (Changeable) to immutable(Not changeable).

                        IEnumerable ToImmutableList(this IEnumerable sourceList);Example:
                        IList list1 = new List{2, 4, 6, 8};   
IList immutableList = list1.ToImmutableList();

ReplaceWith

Replaces a list items with another ones items.

                        IEnumerable ReplaceWith(this IList sourceList, IEnumerable targetList);Example:
                        IList list1 = new List{2, 4, 6, 8};   
IList list2 = new List(); 
list2.ReplaceWith(list1);

PickRandomItem

Picks one item through the list randomly..

                        T PickRandomItem(this IList sourceList);Example:
                        IList list1 = new List{2, 4, 6, 8};   
int randomItem = list1.PickRandomItem();
